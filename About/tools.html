Below is a list of available tools, frameworks, Software Development Kits (SDKs), and supported languages for seL4, either provided by the seL4 Foundation or provided by the seL4 community. Some are open source (potentially still in development), while others are commercial. If you would like an item to be added, please contact us at foundation@sel4.systems.

Development Kits & Component Frameworks

{% include card_tool.html title="MicroKit" type="Open Source" foundation=true link="https://docs.sel4.systems/projects/microkit" body='

Abstraction layer, SDK and tools that simplifies development of statically architected seL4-based systems.

' %} {% include card_tool.html title="CAmkES" type="Open Source" foundation=true link="https://docs.sel4.systems/projects/camkes/" body='

Development and runtime component framework for microkernel-based embedded static systems.

' %} {% include card_tool.html title="sDDF" type="Open Source" foundation=false link="https://trustworthy.systems/projects/drivers" body='

A verifiable seL4 device driver framework providing a set of interfaces, designs and tools for developing high-performance, encapsulated device drivers for seL4, developed at UNSW. A Request for Comment (RFC) is open for consideration by the seL4 Foundation.

' %} {% include card_tool.html title="seL4 Dev Kit" type="Open Source" foundation=false link="https://github.com/sel4devkit" body='

An seL4 development kit aimed at helping complete beginners get started and evaluate seL4 for their needs, developed by Capgemini Engineering and funded by the National Cyber Security Centre (NCSC).

' %} {% include card_tool.html title="BriefCase" type="Open Source" foundation=false link="https://github.com/loonwerks/BriefCASE-Tutorial" body='

A model-based systems engineering environment that can build systems targeting seL4, based on AADL and OSATE, from the DARPA CASE program.

' %}

Operating Systems

{% include card_tool.html title="Lions OS" type="Open Source" foundation=false link="https://trustworthy.systems/projects/LionsOS/" body='

Highly modular, adaptable, verifiable, high-performance OS for statically architected embedded/IoT/cyberphysical systems, developed at UNSW.

' %} {% include card_tool.html title="KOS" type="Commercial" foundation=false link="https://www.kry10.com/#platform/" body='

Kry10 Operating System (KOS) and Platform, designed for mission critical connected devices and integrated with the Erlang BEAM application environment, developed by Kry10.

' %} {% include card_tool.html title="Genode" type="Open Source" foundation=false link="https://genode.org/" body='

Genode is an operating system framework for hierarchical dynamic systems based on L4-family microkernels, including seL4.

' %}

Virtualisation

{% include card_tool.html title="VM Composer" type="Commercial" foundation=false link="https://www.dornerworks.com/vm-composer" body='

A Secure Microkernel Virtual Machine (VM) Composer, a modeling tool to develop and deploy virtualised high-assurance systems using a drag-and-drop interface, developed by DornerWorks.

' %}

Language Support

C is the default seL4 language, but other languages are supported, in particular for application-level components.

{% include card_tool.html title="Rust" type="Open Source" foundation=true link="https://github.com/seL4/rust-sel4" body='

Comprehensive support for using the Rust programming language to write seL4 applications.

' %} {% include card_tool.html title="Pancake" type="Open Source" foundation=false link="https://trustworthy.systems/projects/pancake/" body='

Low-level, imperative language with a verified compiler, designed for implementing verifiable device drivers and other OS code.

' %}
