Below is a list of available tools, frameworks, Software Development Kits (SDKs), and supported languages for seL4, either provided by the seL4 Foundation (in green) or provided by the seL4 community (in blue). Some are open source (potentially still in development), while others are commercial. If you would like an item to be added, please contact us at foundation@sel4.systems.

Development kits & Component frameworks

{% include card.html icon="cloud-arrow-down" title="MicroKit" link="https://docs.sel4.systems/projects/microkit" body='

Abstraction layer, SDK and tools that simplifies development of statically architected seL4-based systems.

' %} {% include card.html icon="cloud-arrow-down" title="CAmkES" link="https://docs.sel4.systems/projects/camkes/" body='

Development and runtime component framework for microkernel-based embedded static systems.

' %} {% include card.html icon="cloud-arrow-down" icon_colour="f_blue" link="https://trustworthy.systems/projects/drivers" title="sDDF" body='

A verifiable seL4 device driver framework providing a set of interfaces, designs and tools for developing high-performance, encapsulated device drivers for seL4, developed at UNSW. A Request for Comment (RFC) is open for consideration by the seL4 Foundation.

' %} {% include card.html icon="cloud-arrow-down" icon_colour="f_blue" link="https://github.com/sel4devkit" title="seL4 Dev Kit" body='

An seL4 development kit aimed at helping complete beginners get started and evaluate seL4 for their needs, developed by Capgemini Engineering and funded by the National Cyber Security Centre (NCSC).

' %} {% include card.html icon="cloud-arrow-down" icon_colour="f_blue" link="https://github.com/loonwerks/BriefCASE-Tutorial" title="BriefCase" body='

A model-based systems engineering environment that can build systems targeting seL4, based on AADL and OSATE, from the DARPA CASE program.

' %}

Operating Systems

{% include card.html icon="cloud-arrow-down" icon_colour="f_blue" link="https://trustworthy.systems/projects/LionsOS/" title="Lion OS" body='

Highly modular, adaptable, verifiable, high-performance OS for statically architected embedded/IoT/cyberphysical systems, developed at UNSW.

' %} {% include card.html icon="building-storefront" icon_colour="f_blue" link="https://www.kry10.com/#platform/" title="KOS" body='

Kry10 Operating System (KOS) and Platform, designed for mission critical connected devices and integrated with the Erlang BEAM application environment, developed by Kry10.

' %} {% include card.html icon="building-storefront" icon_colour="f_blue" link="https://www.trentos.de" title="TRENTOS" body='

An seL4- and CAmkES-based statically architected OS and SDK for embedded systems, developed by HENSOLDT Cyber.

' %} {% include card.html icon="cloud-arrow-down" title="CantripOS" link="https://github.com/AmbiML/sparrow-cantrip-full" body='

An operating system written almost entirely in Rust, that runs on top of seL4 using CAmkES, aimed at low-power secure embedded platform for Ambient ML applications, developped by Google.

' %}

Virtualisation

{% include card.html icon="building-storefront" icon_colour="f_blue" link="https://www.dornerworks.com/vm-composer" title="VM Composer" body='

A Secure Microkernel Virtual Machine (VM) Composer, a modeling tool to develop and deploy virtualised high-assurance systems using a drag-and-drop interface, developed by DornerWorks.

' %}

Language Support

C is the default seL4 language, but other languages are supported, in particular for application-level components.

{% include card.html icon="cloud-arrow-down" title="Rust" link="https://github.com/seL4/rust-sel4" body='

Comprehensive support for using the Rust programming language to write seL4 applications.

' %} {% include card.html icon="cloud-arrow-down" icon_colour="f_blue" link="https://trustworthy.systems/projects/pancake/" title="Pancake" body='

Low-level, imperative language with a verified compiler, designed for implementing verifiable device drivers and other OS code.

' %}
