seL4 is available for a wide range of platforms on Arm, RISC-V, and Intel architectures. The kernel is intended to run on hardware ranging from small IoT boards to server class machines, with current focus on the embedded space.

' layout: card ---

The following hardware is supported:

  • Arm: Avnet MaaXBoard, BeagleBoard, BeagleBone Black, IMX8MM-EVK, Odroid-C2, Odroid-C4 OdroidX, OdroidXU, OdroidXU4, Raspberry Pi 3b, Raspberry Pi 4B, Rockpro64, Sabre Lite, TK1, TK1-SOM, TX1, TX2, Ultra96v2, Zynq ZCU102, Zynq-7000, IMX8MQ
  • RISC-V: Ariane, Cheshire, HiFive Unleashed, Microchip PolarFire, Rocketchip
  • Intel: 32 and 64 bit PC-99 machines

Virtualisation and SMP multicore support is available on most of these boards. For details on features and verification status of the kernel on these boards, see the extensive hardware page on the seL4 docsite.

Qemu simulator support is available for all of the architectures above.
